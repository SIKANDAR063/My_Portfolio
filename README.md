## 📌 Features

### ✨ Core Highlights
- ⚡ Lightning-fast performance (optimized CSS & JS)
- 🌙 Dark / Light theme with system preference support
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎯 Smooth scroll, parallax & scroll-triggered animations
- 🧩 Modular ES6 JavaScript architecture
- 🎨 3D Galaxy background with Three.js
- 🌀 Rotating tech icons with animations
- 📩 Contact form with client-side validation

## 🚀 Installation & Setup

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)
- Git (optional, for cloning)

### Method 1: Direct Download
1. Download the ZIP file from GitHub
2. Extract to your desired folder
3. Open `index.html` in your browser

### Method 2: Clone Repository
```bash
git clone https://github.com/SIKANDAR063/sikandar-portfolio.git
cd sikandar-portfolio

# Install Live Server extension in VS Code
# Right-click on index.html
# Select "Open with Live Server"


# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Then open http://localhost:8000


# Install globally
npm install -g http-server

# Run server
http-server -p 8080

# Open http://localhost:8080# My_Portfolio
